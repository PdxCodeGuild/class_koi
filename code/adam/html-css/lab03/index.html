Sub-Orbital

A sub-orbital spaceflight is a spaceflight in which the spacecraft reaches outer space, but its trajectory intersects the atmosphere or surface of the gravitating body from which it was launched, so that it will not complete one orbital revolution (it does not become an artificial satellite) or reach escape velocity. This spaceflight will show the curvature of the earth but not a view of its entirety.
